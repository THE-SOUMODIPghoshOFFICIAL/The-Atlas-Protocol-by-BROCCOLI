## Game Dev Roadmap

<details>
<summary><strong>Beginner â†’ Intermediate â†’ Advanced (click to expand)</strong></summary>

### Phase 1: Foundations

- Game loop, event handling, input systems
- Sprites, animation, collision detection
- Basic scene/world management
- 2D top-down or side-scrolling game (first project)

### Phase 2: Intermediate

- Game state machines, UI systems, menus
- Physics engines and rigidbodies
- Particle systems and visual effects
- Audio: music, SFX, mixing basics
- Level design and progression

### Phase 3: Advanced

- 3D graphics and lighting
- Advanced physics, ragdoll systems
- Networking (multiplayer, client-server)
- AI and pathfinding (nav meshes, behavior trees)
- Performance profiling and optimization
- Build pipelines and publishing (Steam, mobile, console)

</details>
